# ООО "Обувь" - Информационная система магазина обуви

## Описание проекта

Веб-приложение для управления магазином обуви, разработанное на Python с использованием фреймворка Flask.

## Функциональность

### Модули системы:

1. **Модуль авторизации**
   - Вход по логину и паролю
   - Вход как гость (ограниченный доступ)
   - Отображение ФИО пользователя в правом верхнем углу

2. **Роли пользователей:**
   - **Гость** - просмотр товаров без фильтрации/сортировки/поиска
   - **Авторизованный клиент** - просмотр товаров без фильтрации/сортировки/поиска
   - **Менеджер** - просмотр товаров (с фильтрацией, сортировкой, поиском), просмотр заказов
   - **Администратор** - полный доступ: добавление/редактирование/удаление товаров и заказов

3. **Список товаров**
   - Отображение фото, артикула, наименования, категории, производителя, поставщика
   - Отображение цены (перечеркнутая красная цена при скидке, черная итоговая)
   - Подсветка строк:
     - Зеленый (#2E8B57) - скидка более 15%
     - Голубой - товара нет на складе

4. **Управление товарами (администратор)**
   - Добавление нового товара
   - Редактирование товара
   - Удаление товара (нельзя удалить товар, который есть в заказах)
   - Загрузка фото товара (автоматический ресайз до 300x200)

5. **Управление заказами (менеджер/администратор)**
   - Просмотр списка заказов
   - Просмотр детальной информации о заказе
   - Добавление/редактирование/удаление заказов (администратор)

## Цветовая схема (согласно руководству по стилю)

- **Основной фон:** #FFFFFF (белый)
- **Дополнительный фон:** #7FFF00 (лаймовый)

- **Скидка >15%:** #2E8B57 (зеленый)
- **Нет на складе:** #87CEEB (голубой)

## Установка и запуск

### 1. Установка Python

Убедитесь, что у вас установлен Python 3.8 или выше:
```bash
python --version
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск приложения

```bash
python app.py
```

### 4. Открытие в браузере

Перейдите по адресу: http://localhost:5000

## База данных

База данных SQLite содержит следующие таблицы:
- `roles` - роли пользователей
- `users` - пользователи
- `categories` - категории товаров
- `manufacturers` - производители
- `suppliers` - поставщики
- `products` - товары
- `pickup_points` - пункты выдачи
- `order_statuses` - статусы заказов
- `orders` - заказы
- `order_items` - позиции заказов

## Хостинг с локального ПК

Для доступа к приложению из локальной сети:

1. Измените в `app.py` строку:
```python
app.run(debug=True, host='0.0.0.0', port=5000)
```

2. Запустите приложение

3. Откройте порт 5000 в брандмауэре (если необходимо)

4. Доступ по адресу: `http://<IP-адрес-ПК>:5000`
